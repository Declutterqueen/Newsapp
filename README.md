# 📰 NewsApp (React)

A modern news web application built with **React** that fetches and displays the latest news articles using a news API.

👉 **Live Demo:** https://newsapp-six-theta.vercel.app/

---

## 🚀 Features

- 🗞️ Latest news articles
- 📂 Category-based news (e.g., Business, Sports, Technology)
- ⚛️ Built with React (Create React App)
- 📱 Responsive design
- 🔄 Dynamic content via API

---

## 🛠 Tech Stack

- React.js
- JavaScript (ES6+)
- CSS3
- News API
